GitHub Pages / Rust / WASM

lingo

Rust + WASM で動く多言語翻訳チャットアプリの公開ページ。 まずは GitHub Pages の入口、CSS 分離、WASM 読み込み口までを揃えています。

翻訳デモへ
Pages

公開ルートを docs/ に設定

GitHub Pages でそのまま配信できる構成にしています。

CSS

スタイルを style.css に分離

HTML は構造だけにして、見た目は独立管理します。

WASM

翻訳処理を Rust/WASM に集約

ブラウザーは入出力だけを担当し、翻訳ロジックは WASM 側に置きます。

Demo

Transformers.js 翻訳チャットの入口

engine: transformers WASM: booting

WASM の本体は `docs/wasm/` 配下に配置します。 ブラウザーは Rust/WASM が返した翻訳結果をそのまま表示します。
